Vinary

Vinary Pair: Community Curated Pairings

HTML | CSS | React.js | Node.js | PostgreSQL

GitHub Repo Live Demo

This application allows users to search a database of recipes by dish or by cuisine and create a beverage pairing for their selected dish. It's for peope who like to enjoy dinner and a beverage and memorialize those special moments, while sharing them with other users and friends.

I created this app because it's just the kind of app I want to remember the less romantic bits of a fantastic dinner.
